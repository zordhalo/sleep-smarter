How Sleep Works: Two Key Systems

System 1: Sleep Pressure (Adenosine)

  • Builds up while you're awake
  • Creates 'sleepiness' sensation
  • Caffeine blocks adenosine receptors temporarily

System 2: Circadian Rhythm (Internal Clock)

  • 24-hour cycle controlled by light
  • Morning light ‚Üí earlier sleep
  • Evening light ‚Üí delayed sleep

The Cognitive Cost:

  • Sleep <6h = 20% ‚Üì memory, 23% ‚Üì concentration
  • Sleep after learning = 20-40% ‚Üë memory retention
